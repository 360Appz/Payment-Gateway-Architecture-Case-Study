# Layer 1: Client & Edge Layer

<br/>

## Client Applications

**Supported Channels:**

- **Web** (React / Vue / Angular with checkout SDKs)
- **Mobile** (iOS / Android native SDKs)
- **Point of Sale (POS)** terminals
- **Server-to-Server** API integrations

<br/>

## CDN & Edge Network

**Technologies:** Cloudflare, AWS CloudFront, Fastly

**Capabilities:**

- Global edge distribution (200+ PoPs)
- SSL/TLS termination
- Static asset caching
- DDoS mitigation (Layer 3 / 4 / 7)

<br/>

## Web Application Firewall (WAF)

**Protection Against:**

- SQL injection
- Cross-site scripting (XSS)
- OWASP Top 10 vulnerabilities
- Bot detection & mitigation
- Rate limiting per IP / user

<br/>

## API Gateway

**Features:**

- Request routing & transformation
- Protocol translation (REST, GraphQL, gRPC)
- API versioning
- Request / response validation
- Circuit breaking
- Rate limiting (tenant-based)

**Rate Limits:**

- **Standard tier:** 100 req/min  
- **Premium tier:** 1,000 req/min  
- **Enterprise tier:** Custom limits
